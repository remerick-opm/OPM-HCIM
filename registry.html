What is the Registry?

The HCIM Registry contains a listing of data elements within the HRLOB data standard that are implementation ready and fit for use. This list is an extract of the Human Capital Information Model (HCIM) and is in alignment with the International Organization for Standardization (ISO) 11179 naming standard.

Recent Updates

Added new data elements for A1 (Agency Human Capital strategy, Policies, and Operation Plan) Updated language in business names for historical data elements without making substantive changes to content Added two new fields to the Registry file to explain when and why certain data elements are deprecated This is a major release. All prior deprecated data elements were removed.
