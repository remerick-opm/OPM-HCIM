Products Description
Domain Values Each month the HRLOB releases the most up-to-date Domain Value Repository. Domain Values and any associated metadata in the repository have been collected, maintained and stored by the HRLOB. The table below displays the past Domain Value releases. The table below contains all current and past Domain Value releases along with the release dates. Please use the table below to download or view the most up to date Domain Value file.
Registry The HCIM Registry contains a listing of data elements within the HRLOB data standard that are implementation ready and fit for use. This list is an extract of the Human Capital Information Model (HCIM) and is in alignment with the International Organization for Standardization (ISO) 11179 naming standard.
Systems and Forms Mapping The Systems and Forms Mapping workbook includes all systems/forms that have been mapped to the HCIM. To be "mapped" refers to the linkage of a system's/form's elements to the HCIM data elements. Items from systems/forms are either linked to existing elements or added to the repository as new elements.

HCIM Release Notes

The Human Capital Information Model (HCIM) builds on HRLOB’s commitment to advance data standardization and modernization using industry-aligned best practices to help agencies and providers in the implementation of HCIM products. This is a proud moment for the HRLOB, for the federal HR community, and for the continual goal of improving government, as we can present both new and improved products that will aid in the implementation of HCIM products and enhance our data standardization efforts.

Below is a listing of the HCIM products that have been updated with the most recent HCIM release:
